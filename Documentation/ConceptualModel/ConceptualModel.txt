1.2 Model Overview
The conceptual model represents a comprehensive book rating and management system with the following key components:
Core Entities:
●	BaseEntity: Abstract base class providing common fields (Id, CreatedAt, UpdatedAt)
●	Book: Central entity containing publication information
●	Author: Represents book authors with biographical information
●	Publisher: Contains publisher details
●	Category: Enables book classification
●	ReviewRating: Handles user reviews and ratings
●	Event: Manages book-related events
●	ApplicationUser: Represents system users
●	Wishlist: Manages user book collections
